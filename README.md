
# Jwt Authentication

A brief description of what this project does and who it's for


## Run Locally

Clone the project

```bash
  git clone https://github.com/Himanshu7487/jwtauth.git
```

Go to the project directory

```bash
  cd jwtauth
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
